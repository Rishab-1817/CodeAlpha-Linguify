# 🌐 LINGUIFY

> **Translate Instantly. Connect Effortlessly.** 
> **Break Boundaries, Not Conversations.**

A futuristic AI-powered translator supporting **100+ languages** with real-time translation, speech support, and a modern cyberpunk-inspired interface.

## ✨ Features

- 🌍 100+ Languages
- ⚡ Real-Time Translation
- 🔍 Auto Language Detection
- 🔊 Text-to-Speech
- 📋 One-Click Copy
- 🔄 Language Swap
- 🎨 Responsive UI
- 📱 Mobile Friendly

## 🚀 Live Demo
https://linguify-svbg.onrender.com


## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- FastAPI
- Python
- Uvicorn

## 📦 Installation

```bash
git clone https://github.com/Rishab-1817/CodeAlpha-Linguify.git
cd CodeAlpha-Linguify
pip install -r requirements.txt
uvicorn main:app --reload
```
